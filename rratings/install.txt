Почему?
Хак сделан так а никак не иначе потому что не хочется залезать в код Инстанта и вносить в него изменения. Единственный файл который нужно будет изменить, это файл темплейта показывающий фотографии "com_photos_view_photo.tpl", который никак не может повлиять на работоспособность Вашего Инстанта. Даже если Вы что-то сделаете не так, Ваш сайт будет работать. 


Инструкция по установке.

Закачайте папку rratings  в корень вашего сайта. Откройте файл _config-rating.php введите данные для вашей базы.
Выполните запрос в базу:
CREATE TABLE `cms_rratings` (
  `id` varchar(11) NOT NULL,
  `total_votes` int(11) NOT NULL DEFAULT '0',
  `total_value` int(11) NOT NULL DEFAULT '0',
  `used_ips` longtext,
  PRIMARY KEY (`id`)
) ENGINE=MyISAM DEFAULT CHARSET=cp1251;

Если у Вас другой перфикса не cms_, измените его и в запросе и в конфигурационном файле.

Далее откройте файл темплейта com_photos_view_photo.tpl
и вставьте туда:
<script type="text/javascript" language="javascript" src="/rratings/js/behavior.js"></script>
<script type="text/javascript" language="javascript" src="/rratings/js/rating.js"></script>
<link rel="stylesheet" type="text/css" href="/rratings/css/rating.css" />
{php}
require('rratings/_drawrating.php');
require('rratings/_config-rating.php');
echo rating_bar('foto'.$this->_tpl_vars['photo']['id'],'');
{/php}
В принципе в любое удобное для Вас место. Если вставить внутри выражения {if $photo.album_nav} {/if} то голосование будет включаться выключаться в админке вместе с навигацией альбома.



Обратите внимание на строку echo rating_bar('foto'.$this->_tpl_vars['photo']['id'],'что тут?');
в ней присутствует перфикс foto. Больше ничего не дописывайте,  ни подчерков ни точек ни пробелов. Скрипт перестанет работать. Это для того что бы можно было индентифицировать за что голосуем. Если захотите поставить этот же скрипт в другое место, обязательно используйте перфикс. Это поможет избежать проблем с выводом рейтинга голосований. Какой перфикс и где использовать можете спросить на форуме http://instant.clubcontact.ru. 
И в конце где написано "что тут?" можно поставить количество звездочек в голосовании. По умолчанию стоит 10. Можно написать любой количество но не больше 10.


Возможно выйдет модуль для вывода рейтингов на любые страницы сайта. Следите за новостями на сайте http://instant.clubcontact.ru
